# Getting Started
## Major Dependencies
| Server | Client|
|--------|-------|
| express| react |
| mongodb| react-router |
|mongoose|cors|
|passport|cookie-session|
|        |jwt   |

## Install the dependencies
```bash
$ npm install
```
## Start Mongo server
```bash
$ sudo service mongod start
```
## Run the Server
```bash
$ cd server
$ node server
```

## Run the React server
```bash
$ npm start
```
