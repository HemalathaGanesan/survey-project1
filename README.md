#  Survey-Project Form Pages 

# Getting Started
## Major Dependencies
| Server | Client|
|--------|-------|
| express| react |
| mongodb| react-router |
|mongoose|cors|
|passport|cookie-session|
|        |jwt   |

## Install the dependencies
```bash
$ npm install
```
## Start Mongo server
```bash
$ sudo service mongod start
$ cd server
$ node server
```
## Run the React server
```bash
$ npm start
```
## Run the Server
```bash
$ node app.js
```
